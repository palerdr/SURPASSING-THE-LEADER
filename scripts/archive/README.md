# `scripts/archive/`

Historical probes and one-off validation scripts.

These files are kept for reference, but they are not active entry points:

- `_feas_probe.py` - early LP/invariant timing probe with old local paths.
- `_limit_game_probe.py` - early all-deaths-fatal limit-game script.
- `_preleap_probe.py` - early pre-leap reachability count.
- `validate_interior_fix.py` - one-off validation for an interior-balancing fix.
- `train_gate_wd.py` - one-off weight-decay sweep/gate script.

Prefer the active scripts in the parent directory for new runs.

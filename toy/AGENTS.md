# Toy Project Instructions

This subtree is a small, fully enumerable demonstration—not a second STL
engine. Keep one canonical fixed-bucket ruleset, deterministic artifact
manifests, and exact LP/tablebase certificates.

- Never import `stl` or `dth`.
- Do not add leap seconds, route stages, private information, or production
  checkpoint compatibility.
- Generated files belong under `toy/outputs` or `toy/checkpoints`.

Run `uv run python -m pytest toy/tests -q`.
